# StudySync Landing Page

A modern and responsive educational landing page built using HTML5 and CSS3.

## Features

* Responsive Navigation Bar
* Hero Section
* Company Showcase
* Feature Cards
* Testimonials Section
* Newsletter Subscription Form
* Footer with Social Links
* Mobile-Friendly Layout

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* SVG Icons
